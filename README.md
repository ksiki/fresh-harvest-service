# 🌱 «Свежий урожай»

## 1. Общие сведения

**Название продукта:** «Свежий урожай»  
**Платформа:** Telegram Mini App (TMA)

### Целевая аудитория

**Продавцы:**
- Частные лица  
- Дачники  
- Фермеры (включая лиц пожилого возраста)

**Покупатели:**
- Городские жители  
- Путешественники  
- Дачники-соседи  


## 2. Цели и задачи

Создание удобного и доступного инструмента для прямой коммуникации между производителями сельхозпродукции и потребителями.  

Основной упор делается на:
- Простоту использования (Elderly-friendly UX)  
- Геопозиционирование  

## 3. Технологии

- **python**
- **fastapi**
- **aiogram**
- **postgreSQL + postGIS**
- **yandex maps API**
- **docker compose**
- **poetry**
